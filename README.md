# Trivia Trainer

Mobile-friendly trivia practice app built with React, Vite, shadcn, and Tailwind CSS v4.

## Stack

- React 19
- Vite 7
- shadcn/ui
- Tailwind CSS v4
- Biome for formatting and linting

## Features

- Mix and match quiz categories
- Nearly 500 questions, including full world-country coverage and major rivers
- Randomized question order every round
- Immediate answer feedback with explanations
- End-of-round review for missed questions
- Layout tuned for phones first and larger screens second

## Run

```bash
npm install
npm run dev
```

## Quality checks

```bash
npm run format
npm run lint
npm run build
```

## Regenerate World Data

```bash
npm run generate:trivia
```
